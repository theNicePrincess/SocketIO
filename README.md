什么是 Socket.io 
socket.io 是一个库，可以用于浏览器和服务器之间进行实时。双向和基于事件的通信。它包括：
  使用node.js服务器
  为浏览器（可从node.js的也运行） 一个JavaScript客户读库


自动重新连接支持

断线检测

二进制支持

多路传输支持

客房支援

正在安装

  服务器
  npm install --save socket.io

  javascript 客户端
  npm install --save socket.io-client
